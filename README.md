# 💪 Fitness Tracker System

## 📌 Overview
A web-based fitness management system that helps users track their diet, workouts, calories, and BMI with personalized recommendations.

---

## 🚀 Features
- 🔐 User Registration & Login
- 🥗 Diet Plan Management
- 🏋️ Workout Tracking
- 🔥 Calorie Monitoring
- 📊 BMI Calculator
- 🤖 Personalized Diet & Workout Recommendations

---

## 🛠 Tech Stack
- Frontend: React.js
- Backend: Django REST Framework
- Database: MySQL

---

## 📂 Modules
- User Management
- Diet Management
- Workout Management
- BMI & Recommendation System

---
